Nosso grupo desenvolveu o Versos & Encontros, um sistema web responsivo criado para incentivar a interação entre alunos por meio da leitura. A plataforma permite que os usuários escrevam e compartilhem análises críticas de livros e artigos, promovendo um espaço dinâmico de troca de ideias.

Com interface simples, acesso por qualquer dispositivo e organização intuitiva dos temas, o projeto busca aumentar o engajamento, estimular o hábito da leitura e fortalecer a interação entre estudantes no universo literário.

Benefícios principais:

Incentivo à leitura

Maior engajamento com obras literárias

Espaço colaborativo para discussão e construção do conhecimento

Foi uma experiência enriquecedora transformar uma ideia em um ambiente que promove aprendizado e troca real entre alunos.
